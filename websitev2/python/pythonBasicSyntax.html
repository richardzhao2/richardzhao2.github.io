Introduction

Before we get started, you're going to need to install Python and a text editor and/or IDE.

You can go to python's website here and download Python 3. You will also need a text editor or IDE; I personally use Sublime because it's very lightweight and fast. However, setting up Sublime for Python can be very annoying. Here is a good tutorial for setting up Sublime.

Other easier to deal with options are:

  • Pyzo
  • PyCharm (I'd love to use this one if it were as fast as Sublime)
  • Atom (this is the most aesthetic IMO but it's really slow)
  • Spyder(from Anaconda)
    •

    Side note: Anaconda is just Python with a bunch of packages (extra stuff to help you program with). To use Anaconda, you want to change your path and make it so that you compile programs with Anaconda's version of Python rather than the one you install from python.org.
