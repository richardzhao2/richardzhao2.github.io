    There are 3 main parts to this very simple program. Namely, they are:

  • The "print" part indicates that we are printing something to the console. The general form is print(whatever we want to print)
  • The parenthesis indicate that whatever print intends to print must be within those parenthesis. E.g print("hello")world will not print "hello world" (this will just crash, but pretend it didn't)
  • The quotations indicate that we are printing a string, and whatever's between the quotations to the console. That is to say, print("HELLO") will print "HELLO" to the console. We'll discuss strings indepth later on.

    • And if for instance, I wanted to print out some other string, I would change the contents between the quotation. E.g if I typed in print("Goodbye, cruel world"), the console would print out "Goodbye, cruel world" without the quotations.
