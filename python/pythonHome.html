Computer Science with Python

Hi, my name's Richard and this is a very simple introduction to computer science using Python.

One of the main issues I have with a lot of online resources is that they're usually glorified syntax references. Rarely do I find sources that develop problem-solving skills, and the ones that do often focus very heavily on that alone, so I made this site in hopes that it'll help bridge the gap between knowing a bunch of syntax and knowing how to use that knowledge to solve problems.

Another thing I hate about a lot of online resources is that they always only use videos(which are nice in their own way) but sometimes I want to power through documentation and learn quickly instead of listening to poorly written jokes for an extra 3 minutes. Other times I like to watch a video when I need to go slower. Sometimes I like to do practice problems to brush up on skills, and other times I just want a refresher and want to look at a bunch of examples of implementation, so ideally I'd like to have all 3 (videos coming soon!)

I added a solutions section to give a sample solution of the problems. There are multiple ways to solve each problem and there's definitely better coded ones than the ones I provide. A lot of websites don't like giving you solutions in hopes that it "forces" you to learn, or they don't explain what their code actually does (either no explanation or comments) but what usually ends up happening is I get mad and just leave the site.

I like starting off with Python because it's an easy language to pickup, but language choice doesn't really matter in the end.

You'll note that I don't actually run a Python compiler within the web browser- this is for several reasons. Some of them are actual reasons, and some of them...are questionable. Originally, I wanted to write this as a supplementary source for learning for college students taking their first CS course, and from my experience they usually force you to write code outside of the browser. Another (admittedly bad) reason is that I'm quite too lazy to actually implement such a feature. (Python isn't Javascript!)

Ideally, you'd learn more about problem solving skills so you can switch to any language you want, but hopefully, you'll end up learning these topics:

  • Basic Syntax and Using Operators
  • Functions
  • Conditionals
  • Loops
  • Strings
  • Lists
  • Classes + Object Oriented Programming
  • Recursion
  • Thinking Efficiently
